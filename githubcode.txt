         
                                                      Local area repository 


git init  (git enable in folder)

git config --global user.name " user_name"

git config --global user.email "email"

git add model.h5  (put a file in committed area )

git status    (see the committed area)

git rm --cached model.h5 (get out  file from in committed area )

touch first.html (creating a file which name first.html)

git add *.html (update all to committed html files)

git add .  (update all files to committed in folder)

                                                           commit

git commit -m 'test_01' (files where in commit area update to permanent.."test 01 is the name of commit area" )

git log (you can see the commit list)


                                                             branch

git branch (list of all branch)

git branch login (create a branch call "login")

git checkout login ( switch to branch call "login")



ls (to see the all files in folder)
clear (clear the command lines)




                                         remote repository 

git remote add origin https://github.com/thilanka-sachith/Emotion-Detection-robot.git (මොන  repository එකටද  update කරන්ඩ ඕනේ )

git push -u origin main (update to remote repository everything in 'main' branch )
